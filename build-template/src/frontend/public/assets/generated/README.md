# Generated Assets

This directory contains generated images and audio files for the Temple of Productivi-Tea application.

## Required Assets

The following assets should be placed in this directory:

### Images
- `ancient-border-pattern-tile.dim_1024x1024.png` - Border decoration pattern
- `parchment-texture-bg.dim_2048x2048.jpg` - Background parchment texture
- `curtain-fabric-texture.dim_1024x1024.jpg` - Curtain intro texture
- `jesus-meme-popup.dim_512x512.jpg` - Jesus character popup
- `satan-meme-popup.dim_512x512.jpg` - Satan character popup
- `kanye-meme-popup.dim_512x512.jpg` - Kanye character popup
- `trump-meme-popup.dim_512x512.jpg` - Trump character popup

### Audio
- `religious-background-loop.mp3` - Background ambient audio
- `choir-sting.mp3` - Choir sound effect

## Placeholder Mode

If assets are missing, the application will gracefully handle their absence:
- Images will show alt text or fallback colors
- Audio will be skipped if files don't exist
- The app remains fully functional without media assets
